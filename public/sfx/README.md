Drop custom MP3 sound effects in this folder and the app will use them after the first user interaction.

Supported filenames:

- complete.mp3
- drop.mp3
- seal.mp3
- review.mp3
- pause.mp3
- recover.mp3
- reward.mp3
- spend.mp3
- open.mp3
- error.mp3
- reset.mp3
- tick.mp3
- edit.mp3

If a file is missing, the app uses a quiet built-in Web Audio fallback for that interaction.
