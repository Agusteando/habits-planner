# Habit Planner RPG v9

Focused updates:

- Search added to the Action Bank.
- Dedicated Action Bank import/export/template UI added inside the left rail.
- Action Bank export downloads only action templates, not the whole plan.
- Action Bank import merges valid actions into the existing bank.
- Template download provides a ready-to-edit JSON structure.
- Kanban block cards are larger and easier to read.
- Day columns support vertical scrolling when many blocks are stacked.
- The week board remains one horizontal row.
- The Action Bank remains on the left rail on desktop.

Run:

```bash
npm install
npm run dev
```
