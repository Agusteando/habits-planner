
# Habit Planner RPG v19

Updates in this pass:

- Time view is now a full-week Kanban-style time view, so all visible days remain on screen.
- Time view uses a Y-axis with compressed time bands and seven day columns.
- The current-time laser effect stays in the relevant day/time cell and energizes the cards it crosses.
- Sealed/locked days now get a distinctive full-column visual treatment in both Blocks and Time views.
- The Week header controls were redesigned into grouped controls: week navigation, Today/Hide past, view toggle, and time filters.
- The default view remains Blocks.
- No package-lock.json is included.

Run:

```bash
npm install
npm run dev
```
